Plataforma de Gestión de Restaurante - versión simple

Contenido:
- sql/schema.sql : crea la base de datos y tablas
- database.php : conexión PDO
- menu.php, crear_menu.php, editar_menu.php, eliminar_menu.php
- reservas.php, crear_reserva.php
- pedidos.php, crear_pedido.php, procesar_pedido.php
- inventario.php, actualizar_inventario.php
- reportes.php

Instrucciones:
1. Copia la carpeta a tu www root (ej: c:\laragon\www\RESTAURANTE_FINAL)
2. Ejecuta sql/schema.sql en tu MySQL (phpMyAdmin o mysql client)
3. Ajusta credenciales en database.php si es necesario
4. Abre http://localhost/RESTAURANTE_FINAL/menu.php

Este proyecto es intencionalmente simple para uso educativo.
